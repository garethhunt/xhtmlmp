XHTML-MP Overview

This project allows Mozilla Firefox to render HTML content with the mime-type of application/vnd.wap.xhtml+xml.

For any developer/tester working with applications for mobile developers, this provides an alternative to using a device emulator.

If the application uses device detection to provide correctly rendered content to a device, use the User Agent Switcher extension to test the markup targeted at a specific device.
