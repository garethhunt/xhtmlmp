XHTML-MP Overview

This extension allows Mozilla Firefox to render HTML content with the mime-type of application/vnd.wap.xhtml+xml.

For any developer/tester working with mobile applications, this provides an alternative to using a device emulator. (Although nothing beats testing with actual devices.)

If the application uses device detection to provide correctly rendered content to a device, use the Modify Headers extension to test the markup targeted at a specific device.

XHTML-MP Release

Release 0.3.6 of the XHTML-MP extension is now available.

This release adds support for Firefox 2.0, but will still work with version 1.* of Firefox.
